# 离线包使用教程
本操作先获取root权限后操作，执行下面命令后输入密码就行，输入密码的时候不会有任何显示这点请注意。
```bash
sudo -i
```
### 第一步
这里我们统一把离线包上传到docker文件夹下。然后执行以下命令到docker路径下，路径地址在飞牛里面复制路径获取
```bash
cd /vol1/1000/docker
```
### 第二步
直接安装，如果有是其他版本把镜像包名字改了就行
```bash
sudo dpkg -i fnos-0935-1238.deb
```
### 第三步
重启机器就行，再打开就更新完成了
